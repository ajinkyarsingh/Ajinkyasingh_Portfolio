# Neural Studio Portfolio

Modern personal portfolio built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting Started

Prerequisites: Node.js LTS and npm.

```sh
npm install
npm run dev
```

## Tech Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui

## Build & Preview

```sh
npm run build
npm run preview
```
